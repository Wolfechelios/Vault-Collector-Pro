# WolfeVault 0.8 — Mobile Intelligence

WolfeVault Mobile now imports a verified desktop intelligence snapshot and works fully offline across Capture, Scan Review, Evidence, Search, Smart Collections, and Learning Rules. Mobile decisions are exported as an append-only, checksummed change bundle. Desktop remains authoritative and applies valid changes through SQLite while preserving protected values.

The release adds a native Linux test gate. WebdriverIO drives the packaged Tauri application through `tauri-driver`, exercising real IPC, SQLite bundle exchange, background FTS indexing, and restart persistence. Mobile tests, production builds, Rust tests, and native E2E must all pass before merge.
